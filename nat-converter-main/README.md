# Nat's Macro Converter

> :warning: A new version of the converter is being worked on: [here](https://git.silicate.dev/silicate/siliconv).

This is a web-based macro converter for many different formats.

This project is a subset of [Silicate](https://git.silicate.dev/silicate).
Download Silicate for free [here](https://geode-sdk.org/mods/peony.silicate)!

## Supported file formats

### 2.1
- .replay (OmegaBot 3)
- .replay (OmegaBot 2)
- .replay (OmegaBot 1 - URL)
- .replay (ReplayBot)
- .mhr.json (Mega Hack Replay)
- .mhr (Mega Hack Replay)
- .json (TASBot)
- .echo (Echo Old Json, Echo New Json, Echo New Binary)
- .rush (Rush)
- .kd (KD-Bot)
- .xbot (xBot)
- .zbf (zBot)
- .freplay (Fembot)
- yBot 1
- Plain Text

### 2.2
- .gdr
- .gdr.json
- .ybot (yBot 2)
- .xd (XDBot)
- .slc (v1, v2 and v3)
- .tcm (TCBot)
- Plain Text
